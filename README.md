# 俺は定時で帰りたい！

## 概要

3Dアクションゲームです。プレイヤーは職場を舞台に、上司から追いかけられながらゴミを拾い、定時退社を目指します。制限時間内に全てのゴミを拾うとクリアです。難易度によってゲーム性が変化します。

## ゲームの特徴

・シンプルな操作で遊べる3Dゲーム  
・難易度選択によるゲーム性の変化  
・隠し難易度「ブラック企業」モードを実装   
・ボイスとBGMによる演出  

## 操作方法

・移動：WASDキー または 矢印キー  
・視点操作：マウス  
・ジャンプ：左クリック

## 難易度について

・イージー / ノーマル / ハード / ベリーハード  
・ベリーハードをクリアすると隠し難易度「ブラック企業」が解禁されます  
・ブラック企業モードでは制限時間がなく、上司に捕まるまでゲームが続きます
